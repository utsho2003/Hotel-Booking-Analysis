# Hotel Booking Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on hotel booking data to understand customer behavior, booking patterns, cancellation trends, and pricing strategies. The objective is to analyze hotel performance and identify factors that affect bookings and cancellations using Python and data visualization techniques.

---

## Dataset

The dataset contains hotel booking information such as:

- Hotel type (City Hotel / Resort Hotel)
- Booking cancellation status
- Lead time (days before arrival)
- Arrival date details
- Number of adults, children, and babies
- Customer country
- Market segment and distribution channel
- Average Daily Rate (ADR)
- Customer type
- Reservation status and date

File used: `Hotel Bookings (4).csv`

## Analysis Performed

### Data Cleaning

- Checked for missing values

- Removed duplicate records
- Corrected data types

### Exploratory Data Analysis

- Booking distribution analysis
- Cancellation trend analysis
- Hotel type comparison
- Lead time distribution analysis
- Market segment analysis
- Customer type analysis
- Pricing (ADR) analysis

### Customer & Business Analysis

- Customer behavior analysis
- Cancellation pattern identification
- Revenue-related insights
- Booking source analysis

---

## Key Insights

- City Hotels have more bookings compared to Resort Hotels
- Online travel agencies generate most bookings
- Higher lead time increases cancellation probability
- Most customers are new customers (not repeated guests)
- Pricing varies based on hotel type and booking conditions
